Sistemas Operativos

¿Que es un Sistema Operativo?

Un Sistema Operativo es el software encargado de ejercer el control y coordinar el uso del hardware entre diferentes programas de aplicación y los diferentes usuarios. Es un administrador de recursos de hardware del sistema.

El Sistema Operativo como máquina virtual

Un computador se compone de uno o más procesadores o CPUs, memoria principal o RAM, memoria secundaria (discos), tarjetas de expanción (tarjeta de red, modems y otros), monitor, teclado, mouse y otros dispositivos. O sea, es un sistema complejo. Es cribir programas que hagan uso correcto de todas estas componentes no es una tarea trivial. Peor aún es si hablamos de uso óptimo. Si cada programador tuviera que preocuparse de, por ejemplo, como funciona el disco duro de computador, teniendo además sempre presentes todas las posibles cosas que podrian fall, entonces a la fecha habría escrito una cantidad bastante reducida de programas.

El Sistema Operativo como Administrador de Recursos

La otra tarea del sistema operativo consiste en administrar los recursos de un computador cuando hay dos o más programas que ejecutan simultáneamente y requieren usar el mismo recurso (como tiempo de CPU, memoria o impresora).

Además, en un sistema multiusuario, suele ser necesario o conveniente compartir, además de dispositivos físicos, información.
